## A plugin that no-one asked for? 
 Anyways, here's <s>Wonderwall</s> **Song Preview**!

  Hear a preview of songs in your library as you hover over them, just as you're used to.

---

### Features & Formats
*  **Audio Previews:** Instantly hear snippets just by hovering over your tracks.
*  **Current Format Support:** Only `.psarc` files are supported right now.
*  **Future Roadmap:** Looking into adding support for the `.sloppak` format down the road.
